= LibMSC

== Conventions

* All public structs are valid when zero-initialized
* Fields of all public structs are meant to be accessed directly, unless directed otherwise
* Public structures are caller-allocated, with internal allocations provided by `struct msc_allocator` objects
* Struct initialization functions are named `*_init`
* Struct deinitialization functions are named `*_deinit`, which zeros the struct
* Fallable functions return an int-compatible value, with `0` indicating success
* If a function fails, no input is touched
* Infallable functions return `void`
* Deinitializers never fail
* Parameters follow input -> output order, except:
** `self` or similar arguments are always the first
** `allocator` is always the last
